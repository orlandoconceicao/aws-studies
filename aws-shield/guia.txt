O QUE É AWS SHIELD?

AWS Shield é o serviço de proteção contra ataques DDoS.

DDoS é um ataque onde milhares ou milhões de requisições
tentam derrubar uma aplicação.

============================================================
TIPOS DE SHIELD
============================================================

SHIELD STANDARD
- Gratuito
- Proteção automática
- Layer 3 e 4

SHIELD ADVANCED
- Pago
- Proteção avançada
- Inclui suporte DRT
- Proteção contra custos de ataque

CAMADAS DE PROTEÇÃO
============================================================

Layer 3
Proteção contra ataques de rede (UDP Flood)

Layer 4
Proteção contra ataques de conexão

Layer 7
Ataques HTTP Flood
Necessita WAF + Shield Advanced

============================================================
SHIELD VS WAF
============================================================

Shield -> Proteção DDoS
WAF -> Proteção de aplicação

Ambos juntos = arquitetura segura
