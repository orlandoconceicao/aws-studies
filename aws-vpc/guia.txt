O QUE É VPC?

VPC (Virtual Private Cloud) é uma rede privada virtual
dentro da AWS.

Permite isolar recursos e controlar tráfego de rede.

============================================================
COMPONENTES PRINCIPAIS
============================================================

VPC
Rede principal.
Exemplo CIDR: 10.0.0.0/16

SUBNET
Divisão da VPC.
Pode ser:
- Public
- Private

INTERNET GATEWAY
Permite acesso à internet.

NAT GATEWAY
Permite que instâncias privadas acessem internet
sem ficarem expostas.

ROUTE TABLE
Define para onde o tráfego é direcionado.

SECURITY GROUP
Firewall de instância.

NACL
Firewall de subnet.

============================================================
ARQUITETURA PROFISSIONAL
============================================================

Internet
    ↓
Internet Gateway
    ↓
Public Subnet (Load Balancer)
    ↓
Private Subnet (API / EC2)
    ↓
Private Subnet (RDS)

Banco nunca deve ficar público.

============================================================
BOAS PRÁTICAS
============================================================

- Separar subnets públicas e privadas
- Usar NAT Gateway
- Nunca liberar banco para 0.0.0.0/0
- Separar ambientes dev e prod
- Usar Multi-AZ
- Usar Security Groups específicos

============================================================
CONCEITOS IMPORTANTES
============================================================

CIDR
Public Subnet
Private Subnet
Internet Gateway
NAT Gateway
Route Table
Security Group
NACL
Bastion Host
High Availability

============================================================
ERROS COMUNS
============================================================

- Banco em subnet pública
- Permissão global para banco
- Não usar NAT
- Misturar ambientes
- Não planejar CIDR corretamente

============================================================
CONCLUSÃO
============================================================

VPC é a base da segurança de rede na AWS.

Sem VPC estruturada:
Sistema vulnerável.

Com VPC estruturada:
Arquitetura profissional, escalável e segura.
